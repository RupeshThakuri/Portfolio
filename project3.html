#Project 3

Glowing Gradient Button

Idea and description

From a video on youtube I get an idea to have a project where a button is made glowing with gradient when hovred by a mouse.Sound intrestig right? you can see how that works in above video or going through this link too.

To make it more excited I changed some color of the gradient which makes it more shiny. Have a look!

Here is the HTML code.

                
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<link rel="stylesheet" href="glowinGradientButton.css">
<title>glowinGradientButton</title>
</head>
<body>
<a href="#">Button</a>
</body>
</html>

Here is the CSS code

                
*{
    padding: 0;
    margin: 0;
}
a{
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%,-50%);
    background: linear-gradient(90deg, #11c028, #3d0eaa, #ffeb3b, #db091b);
    width: 200px;
    height: 60px;
    text-decoration: none;
    color: black;
    font-size: 24px;
    text-align: center;
    line-height: 60px;
    text-transform: uppercase;
    background: linear-gradient(90deg, #11c028, #1ed69f, #ffeb3b, #6b9607);
    box-sizing: border-box;
    background-size: 400px;
    border-radius: 30px;
    z-index: 1;
}
a:hover{
    animation: animate 8s linear infinite;
}
@keyframes animate{
    0%{
        background-position: 0%;
    }
    100%{
        background-position: 400%;
    }
}
a:before{
    content: '';
    position: absolute;
    top: -5px;
    left: -5px;
    bottom: -5px;
    right: -5px;
    background: linear-gradient(90deg, #11c028, #1ed69f, #ffeb3b, #6b9607);
    z-index: -1;
    background-size: 400px;
    border-radius: 40px;
    opacity: 0;
}
a:hover::before{
    filter: blur(20px);
    opacity: 1;
    animation: animate 8s linear infinite;
}
