#Project 1

Flipping card hover effect.

Idea and description

From a video on youtube I get an idea to have a project where a card is flipped on hovering. Sound intrestig right? you can see how that works in above video or going through this link too.

To make it more excited we can see the social media icons which is shown at the back when the card is flipped.

Here is the HTML code.

Here is the CSS code

                    
.card{
    position: absolute;
    transform: translate(-50%,-50%);
    left: 50%;
    top: 50%;
    height: 300px;
    width: 400px;
    transition: 0.5s;
    transform-style: preserve-3d;
    perspective: 800px;
}
.card .front img{
    height: 300px;
    width: 400px;
}
.card .front{
    position: absolute;
    width: 100%;
    height: 100%;
    top: 0;
    left: 0;
    background: #000;
    transform: rotateX(0deg);
    transition: 1s;
    backface-visibility: hidden;
}
.card:hover .front{
    transform: rotateX(180deg); 
}
.card .back{
    position: absolute;
    width: 100%;
    height: 100%;
    top: 0; 
    left: 0;
    background: #000;
    transform: rotateX(180deg);
    transition: 1s;
    backface-visibility: hidden;
}
.card:hover .back{
    transform: rotateX(0deg); 
}
.back{
    color: white;
    text-align: center;
}
.back h2, .back span{
    margin-top: 90px;
}
.social-icons a{
    color: white;
    margin: 10px;
    padding: 10px;
    background-color: gray;
    border-radius: 50%;
}
.back span{
    font-size: 16px;
    color: gray;
}
