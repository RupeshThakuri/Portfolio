#Project 2

Glowing Dots On Hover

Idea and description

From a video on youtube I get an idea to have a project where some dots are made glowing when hovred by a mouse.Sound intrestig right? you can see how that works in above video or going through this link too.

To make it more excited I changed some of the border shadow.

Here is the HTML code.

                
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="glowingdotsonhover.css">
    <title>Document</title>
</head>
<body>
    <ul>
        <li>1</li>
        <li>2</li>
        <li>3</li>
        <li>4</li>
        <li>5</li>
    </ul>
</body>
</html>

Here is the CSS code

                

body{
    background: gray;
}
ul{
    display: flex;
    position: absolute;
    transform: translate(-50%,-50%);
    padding: 0;
    top: 50%;
    left: 50%;
    margin: 0;
}
ul li{
    list-style: none;
    font-size: 20px;
    margin: 0 50px;
    width: 30px;
    height: 30px;
    line-height: 30px;
    background: rgba(255, 255, 255, .2);
    border-radius: 50%;
    text-align: center;
    color: #000635;
    transition: .5s;
}
ul li:hover{
    background: white;
    box-shadow: 0 0 10px rgba(255, 255, 255, 1),
                0 0 20px rgba(255, 255, 255, 1),
                0 0 30px rgba(255, 255, 255, 1),
                0 0 0 75px rgba(255, 255, 255, 0.05),
                0 0 0 60px rgba(255, 255, 255, 0.05),
                0 0 0 45px rgba(255, 255, 255, 0.05),
                0 0 0 30px rgba(255, 255, 255, 0.05),
                0 0 0 15px rgba(255, 255, 255, 0.05);
}
